Introduction

HTML stands for Hyper Text Markup Language. But why do we need it? Well HTML is the standard language for creating Web pages.

But what does it actually do? HTML describes the structure of a Web page, it consists of a series of elements and it tells the browser how to display the content of the page.

Basic Syntax

When starting any and all HTML documents you must start with the declaration of:

But remember all syntax start with: < and end with: >
For example: <DOCTYPE HMTL>

The basic document syntax should look like this:

<!DOCTYPE html>
<html>
<body>
<h1>My First Heading</h1>
<p>My first paragraph.</p>
</body>
</html>

Attributes

  • All HTML elements can have attributes.
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in a name/value pairs like: name="value"

The href Attribute


The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

Example


<a href="https://csumb.edu"a>California State of Monterey Bay</a>

The src Attribute


The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:


Example


<img src="img_monterey.jpg">

Text Formatting

HTML Formatting Elements


Formatting elements were designed to display special types of text:


  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins>- Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

Example


<b>This is bold</b>
<i>This is italic</i>
This is <sub>subscript</sub> and <sup>superscript</sup>

Lists

HTML lists allow web developers to group a set of related items in lists.


Example


An unordered HTML list:

  • Item
  • Item
  • Item
  • Item

An ordered HTML list:

  1. First item
  2. Second item
  3. Third item
  4. Fourth item

Unordered HTML List

An unordered list starts with the <ul> tag. Each list item starts with the </li> tag.
The list items will be marked with bullets (small black circles) by default:


Example

<ul>
<li>Coffee</li>
<li>Water</li>
<li>Milk</li>
<li>Tea</li>
</ul>

Ordered HTML List

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.
The list items will be marked with numbers by default:


Example

<ol>
<li>Coffee</li>
<li>Water</li>
<li>Milk</li>
<li>Tea</li>
</ol>
